Julia Fractal Demo

In this page I'm using AJAX (instead of a form submission or pure JavaScript) to let the user regenerate Julia-set fractals. When you push the "Generate!" button, an asynchronous request to generate the fractal image is sent to the server which, in turn, responds with image in PNG format.
