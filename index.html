Welcome!

The purpose of this site is to demonstrate to prospective employers a basic knowledge of various web-development technologies including HTML/CSS, JavaScript, AJAX, PHP and MySQL. These initial examples are simple, but I hope to add more and more sophisticated pages as time goes by.

  • Guest Book -- Sign the guest book while you're here. It's stored in an SQL database.
  • Maze Demo -- Generate mazes with JavaScript.
  • Lone Peak Hike Report -- Flip through hike pictures with PHP.
  • Julia Fractal Demo -- Generate Julia fractals with AJAX.

I'm meaning to provide a CSS sheet for all pages of this site, but have yet to get around to it yet.
